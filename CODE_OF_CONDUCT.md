# 行为准则

## 我们的承诺

Mira 致力于为每一位参与者提供友好、安全、包容的环境，无论经验水平、背景、身份或技术取向。我们期望所有参与者在交流中保持尊重、具体与耐心。

## 预期行为

- **保持尊重**：尊重不同观点与经验差异，不人身攻击。
- **保持具体**：讨论聚焦技术问题本身，提供可操作的信息。
- **保持耐心**：理解开源项目的维护者与贡献者以业余时间参与，给予合理的响应预期。
- **保持诚实**：如实描述问题与能力边界，不夸大也不隐瞒。
- **保护隐私**：不公开他人设备的私有标识符、序列号或个人信息。

## 不可接受的行为

以下行为不被接受，维护者有权移除相关内容或限制参与：

- **骚扰与歧视**：基于性别、性取向、残疾、外貌、种族、宗教、国籍或其他身份特征的骚扰或歧视性言论。
- **威胁与人肉搜索**：威胁、恐吓、或公开他人的私人信息（doxxing）。
- **性化言行**：在项目空间中发表性化或具有性暗示的内容。
- **破坏性互动**：恶意刷屏、故意误导、灌水或干扰正常讨论的行为。
- **公开他人设备的私有标识符**：包括但不限于完整序列号、唯一设备标识等。

## 适用范围

本准则适用于所有项目空间，包括 GitHub Issues、Pull Requests、代码评审、文档讨论以及任何代表 Mira 项目的公开场合。

## 举报

如发现违反本准则的行为，请通过仓库所有者已验证的 GitHub 联系方式私下举报。举报信息将保密处理。

维护者会在合理时间内审查并回应。对于严重违规，维护者有权立即移除内容并限制相关用户的参与权限。

## 归属

本准则受 [Contributor Covenant](https://www.contributor-covenant.org/) 启发，并根据 Mira 项目的实际情况调整。
