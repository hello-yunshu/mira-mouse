# Contributing

Keep device facts, VID/PID values, protocol bytes, model branches, and brand fixtures in the separate plugins repository. Core changes require formatting, lint, tests, a boundary scan, and an evidence-label update. Do not include reverse-engineered source, manufacturer assets, stable identifiers, credentials, or signing keys. Contributions are accepted under the repository's declared licenses.
