# 商标政策

## Mira 商标

"Mira"、"Mira Mouse" 名称及项目徽标（`public/app-icon.png`、`src-tauri/icons/icon.png` 等）为项目所有者持有的商标。

源代码许可证（AGPL-3.0-or-later）授予你使用、修改和再分发代码的权利，但**不授予**使用 Mira 名称或徽标以暗示官方构建、官方背书或与项目存在从属关系的权利。

## 不受限制的使用

以下使用方式不受本政策限制：

- **指示性使用（nominative use）**：在描述性、评论性或兼容性语境中提及 Mira 名称，如"Mira 支持 XXX 型号"或"本插件兼容 Mira"。
- **未修改再分发**：明确标注为未修改的再分发，且不暗示与项目有额外从属关系。
- **兼容性讨论**：在技术讨论中说明与 Mira 的兼容性或集成方式。

## 需要授权的使用

以下使用方式需事先获得项目所有者书面授权：

- 使用 Mira 名称或徽标分发修改后的版本，且可能引起用户混淆的。
- 使用 Mira 名称或徽标推广商品或服务，可能暗示官方关联的。
- 在域名、社交媒体账号或应用商店名称中使用 Mira 商标，可能暗示官方身份的。

## 设备厂商商标

所有设备厂商名称、品牌名称、产品名称及关联徽标均为各自所有者的商标。Mira 项目未获得任何设备厂商的授权、认可或赞助，厂商名称仅用于说明兼容性。

## 联系

如需申请商标使用授权或有商标相关问题，请通过仓库所有者已验证的 GitHub 联系方式沟通。
