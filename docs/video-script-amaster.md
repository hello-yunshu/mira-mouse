# Mira Mouse 视频口播脚本 — 怒喵鼠标功能介绍

---

## 开场

你有没有遇到过这种情况——买了一只高端游戏鼠标，结果发现官方驱动只有 Windows 版，Mac 用户直接被抛弃。

Mira Mouse 就是来解决这个问题的。一个开源、跨平台的鼠标配置工具，macOS 和 Windows 都能用。

今天我们重点来看看，连接怒喵鼠标之后，你都能做什么。

---

## 怒喵鼠标连接后能做什么

怒喵鼠标支持 USB 有线连接和 2.4G 接收器两种方式，插上就能识别。

### 性能调校

首先是性能相关的设置。

DPI 档位——你可以自由切换预设档位，也可以单独修改每一档的数值，精确到你习惯的灵敏度。

轮询率——从 125Hz 到 8000Hz 全覆盖。不管是日常办公还是电竞游戏，都能找到合适的档位。

还有几个进阶选项：按键去抖时间可以调节，防止双击误触；抬升距离，也就是 LOD，控制鼠标抬起来多少就停止追踪；移动同步和角度吸附，这两个对 FPS 玩家特别实用，能让你在快速拉枪时轨迹更稳。

### 电源管理

接下来是电源管理。

你可以分别设置蓝牙和 2.4G 的休眠时间，范围从 10 秒到 18 小时。不用鼠标的时候自动休眠省电，拿起来立刻唤醒。

电池电量、充电状态都是实时显示的，一目了然。

### 灯光控制

灯光方面，怒喵鼠标的灯效支持很全面。

鼠标本体灯光可以调颜色、开关，接收器灯光还能独立控制——选效果、调速度、改亮度，都行。想低调就关掉，想炫酷就拉满。

### 状态信息

你还能实时看到固件版本、当前连接方式、DPI 按键状态，甚至传感器实时 FPS。

按键映射和配置文件切换也支持，在不同场景之间快速切换。

---

## 罗技鼠标呢？

除了怒喵，Mira Mouse 也支持罗技的 HID++ 协议鼠标，包括 DPI 调节、指针速度、报告率、RGB 灯光接管、电池状态、配置文件管理这些常用功能。不过受限于罗技协议本身，像休眠时间、接收器灯光这些暂时还调不了。

---

## 结尾

简单总结一下：Mira Mouse 让你在 Mac 上也能像 Windows 一样，完整配置你的游戏鼠标。怒喵鼠标支持得尤其全面——从性能到灯光到电源，基本上官方驱动能调的东西，这里都能调。
