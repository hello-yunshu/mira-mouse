# Install on macOS

No verified DMG exists yet. The intended unsigned community asset is `Mira_<version>_universal-unsigned.dmg`. Verify its SHA-256, mount it, and drag Mira to Applications. An unsigned build may require Control-click, Open, then explicit Gatekeeper approval. It is not notarized. Universal architecture and Intel launch remain `blocked` until package verification runs.
