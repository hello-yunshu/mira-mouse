# Install on Windows

Download the `Mira_Windows_<version>_x64-setup.exe` community installer from the release assets and verify its SHA-256 before running it. The installer uses Mira-branded NSIS screens, installs for the current user, and places shortcuts in the `Mira` Start Menu folder.

Without trusted code signing, Windows may show SmartScreen or Unknown Publisher. Mira uses the system WebView2 runtime and does not claim Windows ARM64 support.
