# English documentation

This directory contains English translations of selected documents from [`docs/`](../). The project's primary documentation language is Chinese; English is supplementary. [`README.en.md`](../../README.en.md) is the only fully English document and is kept in the repository root.

## Translation policy

Per [`docs/comment-and-doc-style.md`](../comment-and-doc-style.md):

- Chinese is the primary language for all `.md` documents.
- Code blocks, commands, configuration keys, and technical terms remain in English.
- `README.en.md` is the exception with full English content.

Documents not listed below are Chinese-only. Refer to the original Chinese versions under [`docs/`](../) for those.
