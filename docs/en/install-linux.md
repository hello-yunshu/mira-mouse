# Install on Linux

No verified package exists yet. Planned x86_64 assets are AppImage, DEB, and RPM. AppImage does not silently install device permissions; review [Linux permissions](linux-permissions.md). DEB/RPM packages may install only narrow rules generated from the locked bundled plugin. Snap, Flatpak, and Linux ARM64 are not supported targets at this stage.
