# Install on Linux

Mira ships a Linux AppImage artifact. Download `Mira_Linux_<version>_amd64.AppImage` from [GitHub Releases](https://github.com/hello-yunshu/mira-mouse/releases), make it executable, and run it:

```bash
chmod +x Mira_Linux_<version>_amd64.AppImage
./Mira_Linux_<version>_amd64.AppImage
```

AppImage does not silently install device permissions; see [Linux permissions](linux-permissions.md). DEB/RPM packages remain planned artifacts; if released in the future, they will install only the minimal rules generated by the locked-in built-in plugin. Snap, Flatpak, and Linux ARM64 are not supported targets at this stage.
