# Install on Windows

No verified installer exists yet. The intended community asset is `Mira_<version>_x64-setup-unsigned.exe`. Verify SHA-256 before running it. Without trusted code signing, Windows may show SmartScreen or Unknown Publisher. Mira uses the system WebView2 runtime and does not claim Windows ARM64 support.
